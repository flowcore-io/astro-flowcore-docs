1. In the Flowcore Console, navigate to the **Flow Type** that contains your event type.

2. Find the event type you want to update and click **Edit**.
<Image src={EditEventTypeScreen1} alt="Basic Information tab of Edit Event Type dialog" />

3. In the **Basic Information** tab, update:
   - **Description**: Modify as needed.
   - **Enable Sensitive Data**: Toggle on or off.
   - **Sensitive Data Key Path**: Adjust the JSONPath for your identity field.

4. Switch to the **Sensitive Data Configuration** tab to refine your masking schema:
   - Update the **Schema Definition** JSON.
   - Click **Fetch Latest Event** to load real event data.
   - You can see scrambled data to preview how the data will look when scrambled.
<Image src={EditEventTypeScreen2} alt="Sensitive Data Configuration tab in Edit Event Type dialog" />

:::note
  When you update the schema or event data fields, the scrambled data will be updated to show you the effect of the changes.
:::

5. Click **Update Event Type** to apply your changes.
