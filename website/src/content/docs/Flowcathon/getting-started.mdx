Welcome to Flowcathon! This is a platform for building and testing Flowcore flows.

To access the protected content, you'll need to enter the correct access code. Once you've entered the code, you'll be able to view all the protected pages.

(Add any additional instructions or information here)

- create a flowcore account
- install flowcore cli (min version 4.2.0)
- run `flowcore login` and login with your flowcore account
- run `flowcore version` and check that you have the correct version
