Welcome to Flowcathon! This is a workshop where you will learn the basics of the Flowcore platform.

## Communication
We utilise our Discord server for communication during the workshop. Here we will share access credentials necessary to complete the challenges.
You can join the Discord server [here](https://discord.gg/b9c7dRjR).
During the workshop, we will add the participants to a dedicated channel for the specific workshop.

## Prerequisites

<Steps>
<ol>
<li>
To get started, you will need to create a flowcore account. You can do this by going to [flowcore.io](https://flowcore.io) and signing in with your Github account.
</li>
<li>
Install the flowcore cli with the following command:

```sh
npm install -g @flowcore/cli
```

You can find the full installation instructions [here](https://docs.flowcore.io/guides/flowcore-cli/install-cli/).

:::caution
The flowcore cli requires node v20 or higher.
:::
</li>
<li>
Run the following command to login with your flowcore account. 

```sh
flowcore login
```

You can find the full login instructions [here](https://docs.flowcore.io/guides/flowcore-cli/login-cli/).
</li>
<li>
Run the following command to check that you have the correct version.

```sh
flowcore version
```

:::caution
You must have version 4.2.0 or higher.
:::
</li>
</ol>
</Steps>
