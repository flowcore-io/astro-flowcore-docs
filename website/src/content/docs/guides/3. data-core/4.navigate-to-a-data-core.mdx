Press the `Data Cores` button on the navigation panel on the left hand side.
This will take you to the Data Cores page which lists the following:
- all the data cores that are associated with your selected organization
- all the public data cores

<Image src={DataCoresPage}
       alt={"The data cores page, showing the organization's data cores and the public data cores"}/>

You simply press one of the data cores shown to navigate to it. Where you will see the data core's details and the flow types it contains.
